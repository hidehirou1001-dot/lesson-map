# 体験レポート テンプレ

公開前の下書き用テンプレです。
このファイルはサイト導線には載せず、記事化前の整理メモとして使います。

## 使い方

- 事実と感想を分けて書く
- 料金や制度は、実体験だけで断定せず公式情報も確認する
- 強い断定より `〜に向いていそう` `〜は確認しておきたい` の形を基本にする
- 誹謗中傷や、個人が特定される情報は入れない

## 6. 記事化するときの安全チェック

- 事実と感想が混ざっていないか
- 料金や制度を実体験だけで断定していないか
- 一時的な印象を教室全体の特徴として書いていないか
- 他の候補を不必要に下げる書き方になっていないか
- 個人名や個人が特定される情報が入っていないか
