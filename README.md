# LessonMap

愛媛県の習い事・教室比較サイトです。

## 今週やること

1. 検索結果まわりの情報量を整理する
2. 比較メモとお気に入りの役割差をUIで分かりやすくする
3. `title / description / H1` の全体棚卸しを続ける
4. 料金が数値で見える教室データを増やす

## KPI

愛媛県内の地域名検索で迷わず比較できる状態を増やし、`地域入口 -> 検索 -> 記事回遊` の流れを強くする。

## 優先度レイヤー

### P1

- UX改善
- `title / description / H1` の全体棚卸し
- 地域クラスターの完成度引き上げ

### P2

- FAQと構造化データの統一
- 料金データの数値拡充
- 本番前チェックと運用フロー整備

### P3

- 料金の詳細フィルタ
- 計測基盤の整備
- 保守性改善のためのテンプレート化検討
